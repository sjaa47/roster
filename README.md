## Getting Started
To get started with my portfolio, please ensure you have the following prerequisites:
- Python >= 3.7
- Anaconda or Pip for package management

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/sjaa47/roster.git
   ```
2. Navigate to the project directory:
   ```bash
   cd roster
   ```
3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
